# 🚀 Rapido Ride Analytics Dashboard (Power BI)

## 📊 Project Overview
This project presents an interactive **Rapido Ride Analytics Dashboard** built using Power BI.  
The goal is to transform raw ride data into meaningful insights that help in understanding business performance, customer behavior, and operational efficiency.

---

## 🎯 Objectives
- Analyze ride bookings and cancellations  
- Track revenue trends over time  
- Understand service-wise performance  
- Identify popular pickup and drop locations  
- Evaluate payment method usage  

---

## 🔍 Key Features
- 📌 Total Bookings, Revenue & Distance KPIs  
- 📌 Ride Status Analysis (Completed vs Cancelled)  
- 📌 Monthly Revenue Trends  
- 📌 Service-wise Revenue Breakdown (Bike, Auto, Cab, Parcel)  
- 📌 Payment Method Insights (Paytm, GPay, Amazon Pay, QR)  
- 📌 Top Pickup & Drop Locations  

---

## 🛠️ Tools & Technologies Used
- Power BI  
- DAX (Data Analysis Expressions)  
- Data Cleaning & Transformation  
- Data Modeling  
- Data Visualization

## 💡 Key Insights
- Majority of rides are successfully completed with minimal cancellations  
- Bike services generate the highest revenue among all categories  
- Digital payments (UPI-based) are the most preferred payment method  
- Certain locations have higher pickup and drop frequency

## 🚀 How to Use
1. Download the `.pbix` file  
2. Open it using **Power BI Desktop**  
3. Explore the dashboard with filters and visuals
